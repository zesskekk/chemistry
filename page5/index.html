Жұмыс жасау принципі

Аккумулятордың жұмыс жасау принципі

  • Зарядтау кезінде электр энергиясы химиялық энергияға айналады.
  • Разряд кезінде химиялық энергия электр энергиясына айналады.
  • Батареяның жұмысы циклдік

Жүретін реакциялар

    Разряд кезінде(зарядталу кезінде тура осы реакция қарсы бағытта өтеді)

  • Катодтағы реакция

    • PbO2 + SO42- + 4H+ + 2e- -> PbSO4 + 2H2O

  • Анодтағы реакция

    • Pb + SO42- -> 2e- + PbSO4

    Қорғасын сульфаты жеміргеннен кейін су электролизі болады: 2H2O = 2H2 + O2

  • Катодтағы реакция

    • 2H2O + 2e- -> H2 + 2OH-

  • Анодтағы реакция

    • 2H2O -> O2 + 4H+ + 4e-
